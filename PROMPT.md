### 開發要求



- 保持目前專案架構

- 不重新命名函式

- 不修改 API 格式

- 不修改 Google Sheet 欄位

- 不增加不必要套件

- 優先重構既有程式，而非新增重複程式



### 完成交付



完成後請：



1. 說明修改內容

2. 列出修改檔案

3. 進行 Code Review

4. 主動修正發現的問題

5. 確認專案可正常執行後再交付
